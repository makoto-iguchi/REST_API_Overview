ACL (CREATE_NEW_BUCKET, CREATE_NEW_TOPIC)
	Application Scope
		Retrieve the current ACL Entries
			All, /apps/{appID}/acl GET
			With the specified verb, /apps/{appID}/acl/{ACLVerb}

			Verify a user, /apps/{appID}/acl/{ACLVerb}/UserID:{subjectUserID} GET

		Get the template, /apps/{appID}/templates/{templateName}/locales/{locale} GET
		Set or update the template, /apps/{appID}/templates/{templateName}/locales/{locale} PUT
		Delete the template, /apps/{appID}/templates/{templateName}/locales/{locale} DELETE
	Feature Configuration
		Retrieve the current configuration, /apps/{appID} GET
		Modify the configuration, /apps/{appID} POST
