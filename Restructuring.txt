/apps/{appID}/groups/{groupID}/topics	Get	Obtain the list of the available topics in the application, user or group being accessed
/apps/{appID}/groups/{groupID}/topics/{topicID}	Put	Create a new topic. The topicID must be an string following the pattern [A-Za-z0-9_-]* and with maximum length of 64 characters.
/apps/{appID}/groups/{groupID}/topics/{topicID}	Get	Check the topic exists.
/apps/{appID}/groups/{groupID}/topics/{topicID}	Delete	Delete the topic with all the subscriptions it has.
/apps/{appID}/groups/{groupID}/topics/{topicID}/push/messages	Post	Send a push message to the current topic.
/apps/{appID}/groups/{groupID}/topics/{topicID}/push/subscriptions/users	Post	Subscribe the current authenticated user to the topic.
